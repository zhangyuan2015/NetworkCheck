# 检查网络连通性

支持Ping Telnet

配置文件格式
```
[
  {
    "Name": "百度-测试",
    "IP": "www.baidu.com",
    "Port": 80,
    "Timeout": 1000,
    "IsCkPing": true,
    "IsCkTelnet": true
  },
  {
    "Name": "google-测试",
    "IP": "www.google.com",
    "Port": 80,
    "Timeout": 1000,
    "IsCkPing": true,
    "IsCkTelnet": true
  }
]
```
